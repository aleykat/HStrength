# Home Strength Program

A gentle, low-impact, joint-respecting home strength program delivered as a single
self-contained web page, with a built-in session tracker.

- **Live:** https://aleykat.github.io/HStrength/
- Open `index.html` — everything (program, exercises, progression, optional cardio, and the
  tracker) lives in that one file.
- On a phone, use "Add to Home Screen" for an app-like experience.

This page is general fitness guidance, not medical advice.
